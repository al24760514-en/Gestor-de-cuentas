<h1 align="left"> Gestor de cuentas</h1>

###

<p align="left">Un gestor de cuentas el cual guardara las cuentas junto con las contraseñas clasificando por paginas o aplicaciones que uses para hacer mas facil la tarea de tener seguras tu informacion usando contraseñas complicadas, esto se implementara con una base de datos relacional </p>

###

<h2 align="left">Motivacion</h2>

###

<p align="left"> Es muy comun el usar la misma contraseña para todas las cuentas por razones tan sencillas como que es dificil tener contraseñas complicadas en todas tus cuentas y que sean diferentes por lo que es mas sencillo tener a facil acceso pudiendo copiar y pegar sin necesidad de recordar todas tus cuentas.</p>
